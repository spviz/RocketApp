# Features
View information about all SpaceX rockets, including their status, first flight date, and more.
Browse past SpaceX launches, with detailed information about each mission.

# Technologies
- UIKit
- MVP Architecture
- XCTest
- UserDefaults
- URLSession
- Kingfisher
- SwiftLint
# Requirements
The Rocket App requires iOS 13.0 or later to run.
# Acknowledgments
The Rocket App was created using the SpaceX API. I would like to thank SpaceX for providing such a useful and informative API for developers to use.
